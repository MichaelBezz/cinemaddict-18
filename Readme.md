# Проект «Киноман»

***WORK IN PROGRESS***

### Демонстрация проекта:

["Киноман"](https://michaelbezz.github.io/cinemaddict-18) - сервис для фанатов большого кино. Подробная информация о горячих новинках кино, возможность выбрать и сформировать собственный список фильмов к просмотру, обсуждение кинофильмов и многое другое. «Киноман» — поможет провести время интересно.

---

### Основные технологии используемые в проекте:
1. JavaScript
2. Методология ООП
3. Паттерн проектирования MVP

---

### Запустить проект локально
1. Скачать архив с проектом
2. npm ci
3. npm run start
